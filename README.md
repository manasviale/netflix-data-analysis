# netflix-data-analysis
Exploratory Data Analysis of Netflix Movies and TV Shows using Python

## Overview
This project analyzes Netflix content data to uncover trends in content type,
ratings, duration, release years, and country-wise distribution.

## Tools Used
- Python
- Pandas, NumPy
- Matplotlib, Seaborn
- Jupyter Notebook

## Key Insights
- Movies dominate Netflix content, but TV shows are growing.
- Most content is added within a few years of release.
- The US and India are top content-producing countries.
